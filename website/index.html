CLI Reference

Command your development lifecycle.

Terminal 
python main.py agent --category cli_tool --watch

Generate a CLI tool idea, scaffold repo, and watch the development session until completion.

Terminal 
python main.py website --url [URL]

Extract a project idea from a website and initialize the full automation pipeline.

Terminal 
python main.py status [ID] --watch

Monitor an ongoing session and automatically report the Pull Request URL when finished.
